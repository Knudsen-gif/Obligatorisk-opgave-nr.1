
 #Obligatorisk-opgave-nr.1#

LIBRARY MANAGEMENT SYSTEM

Description: 
The program is a python-based system ment for managing items, members & transactions. 
This program was developed to demonstrate understanding of OOP concept such as classes, objects, inheritance & polymorphism.  

Features: 
Inventory Mangement: You are able to add, update & remove items(books & DVDs).
Member Tracking: You are able to Register members, and track which items they currently have borrowed. 
Transaction Logic: The inventory is automated, so when an item is issued or returned form a member the copies will decrease or increase. 
Serach Funktion: Keyword-based searchs across titles, authors & directors. 
Console Output: The console output is formatted as tabular data presentation, which makes it easy to read & clean. This is also done by using f-string alignment.
